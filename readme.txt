To run the command open the directory in the command line and enter:

"make"

-This will compile the program. After this then you will need to enter:

"make run"

-This will execute the program



"make clean"

- This command will remove all the .class files




If you would rather use a compile command to manually run the program you can enter:

javac -cp dist/lib/AbsoluteLayout.jar src/eventr/EventrForm.java



To manually run the program you can enter:

java -cp dist/lib/AbsoluteLayout.jar:src eventr.EventrForm
